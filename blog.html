Project Description: Mini Blog System

This System serves as a vital component within the realm of social media, offering users a platform to express themselves, connect with others, and stay informed.
Can browse more about Github Link

This project offers the following features

1. Users can register by providing their personal information such as name, email, address, and contact number.

2. Allows registered users to write comments on various posts, fostering engagement and facilitating discussions within the community.

3. Promotes engagement through features such as commenting, liking, and sharing.

4. Users can securely logout from their accounts to prevent unauthorized access
